# 臺灣特技舞蹈協會 官方網站

Taiwan Acrobatic Dance Association — Official Website

## 技術架構
- HTML5 + Vanilla CSS + JavaScript
- 中英雙語切換
- GitHub Pages 部署

## 部署說明

### GitHub Pages
1. 將本專案推送到 GitHub repository
2. 前往 Settings → Pages → Source 選 `main` branch
3. 網站將自動發布到 `https://[your-username].github.io/[repo-name]/`

### Zeabur 部署
1. 連接 GitHub repository 至 Zeabur
2. 選擇靜態網站類型
3. 發布網域
